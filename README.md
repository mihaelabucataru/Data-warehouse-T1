# Data-warehouse-T1
Data Engineering Tutorial 1 2025-2026
The naming convention to be followed is snake case. This means that all letters are lowercase and the spaces are replaced with underscores (_). For example, the fact tables will be named fact_[...], and the dimension tables will be named dim_[...].
